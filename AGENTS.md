# Local development notes
- Install dependencies from `src` with `npm install`.
- Start the desktop app from `src` with `npm start` or from the repo root with `.\Run-StandupScribe.ps1`.
- Smoke-test the local API with `npm run verify:api`.
- Build Windows packages from the repo root with `.\Package-StandupScribe.ps1`.
- Use `StandupScribe` for the product brand and `Standup Scribe` only for Windows shell/install surfaces.
